# Archive (nicht Teil des Produkts)

Hier liegen **aufgegebene Design- und Ein-Datei-Experimente**.  
Sie gehören nicht zur öffentlichen Archillator-Oberfläche und werden nicht nach archilles.org synchronisiert.

## Inhalt

| Datei | Was es war |
|-------|------------|
| `legacy/archillator-fancy-design.html` | Frühere „fancy“ Design-Variante |
| `legacy/archillator-old-design.html` | Ältere Single-File-Variante (lokal „old design“) |

## Warum das hier liegt

Damit das Root-Verzeichnis nur noch den **aktuellen** Stand zeigt (`index.html` + `js/`), alte Geschichten aber **auffindbar** bleiben — in Git und im Ordner.

Zum Wiederherstellen einer Datei: einfach aus diesem Ordner kopieren, oder aus der Git-Historie holen.

## Nicht verwenden für

- Live-Seite  
- Tests  
- Demos gegenüber Nutzern  

Source of Truth für das Tool ist das Repo-Root, siehe `docs/STRUCTURE.md`.
